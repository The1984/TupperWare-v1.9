--------------------------------- Instrucciones para hacer uso del sistema ---------------------------------
------------------------------------------------------------------------------------------------------------

1° Ejecutar el script.sql ubicado en: TupperWare-v1.9\src\main\java\script\mysql\TupperWare-ScriptBD.sql
   sobre el motor de base de datos: MySQL v5.7, usuario: root y contraseña: root

2° Ejecutar el sistema TupperWare-v1.9

3° Pedira una contraseña, la cual es: 1234 

------------------------------------------------------------------------------------------------------------
IMPORTANTE: El sistema cuenta con un servicio de envio de mails en cual esta asociado a la cuenta registrada
            al sistema.
	    Para que el sistema funcione con esta funcionalidad, debera dirigirse a Configuracion - Cuenta
	    y hacer los cambios correspondientes. Recordar que la contraseña debe coincidir con la cuenta de
	    Gmail ingresada y que ademas dicha cuenta posea permiso de Acceso de Aplicacion.
	    Esto se logra ingresando a la cuenta, luego a Cuenta - Seguridad y darle Sí a 
	    "Acceso de aplicaciones poco segura".
